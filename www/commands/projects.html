Rubikscube solver

This one has the biggest wow effect in my opinion. Its a rubiks cube bases on simple css transformations. The solve button solves (who would have thought) the cube with the roux method.
The solver itself uses a meet in the middle algorithm or a list of predefined algorithms (like humans do) to create solution maps. Those are precomputed and just applied in this frontend. Check out the GitHub Project and the deployed Website.

Ultimate Tic Tac Toe Website

This project was worked on on two occasions. Implementing the website with a trainable genetic algorihm was my final project for the BMS. Then during my bachelor i extended the training with a particle swarm optimization algorithm. So now you can self train bots with different optimizatino algorithms and compete against them in Ultimate Tic Tac Toe. Try it out here! There are rumors of people beating the hard level, but neither did i do it myself nor did i witnessed anyone else do it anytime.

Mandelbrot explained

In my exchange semester at UW i took a course on data visualization and interactive explainations. The final project was an educational website, which actually won the class votes as best project. So if you want to learn about the Mandelbrot set, check it out at this link. The technology is plain html, css, javascript and web workers. No fancy library, all handcrafted to generate those stunning Mandelbrot visualizations.
